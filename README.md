# Easy SignIn

## Overview

This Project is an open-source project written in ``` Swift ``` and ``` Kotlin ```

The goal of this project is to apply ``` the clean architecture ``` on a module and include basic API operations such as

- Authentication
    - Login
    - Register
    - Reset Password
- Account
    - Fetch Account Info
    - Update Account Info
